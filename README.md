# Endfield Stocks OCR Tool

## About
Uses OCR to get stock data from Arknights: Endfield.<br>
Only works on Windows.


## Installation
Install via pip:<br>
`pip install endfield-stocks-ocr`

Install tesseract:<br>
[Download link](https://github.com/UB-Mannheim/tesseract/wiki)<br>
[Guide](https://tesseract-ocr.github.io/tessdoc/Installation.html)

## Usage
Run `endfield-stocks-ocr` from a cmd.
### If you're not using a 16:9 Monitor, use `endfield-stocks-set-window` first.
